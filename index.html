AngularJS

1. How do you define an AngularJS application? Put "ng-app=appName" in the html tag.

2. What is a controller? It allows for the manipulation of data within the html.

3. What does ng-model do? This binds an input to the JS.

4. What is interpolation markup? The use of angular within html tags

5. What is $scope? The binding between the HTML, the model its bound to, and the controller.

6. What is dependency injection? The dependency of angular objects being defined by the components that are injected into it. E.g. components being injected with factories or services.

7. What is two-way binding? Having an input connected to both the HTML and JS while updating simultaneously.

8. What is 'controller as' syntax? John Papa style angular where you define the controller as a variable in order to make it easier to follow the variables that you define in the controller.

9. What is MV* architecture? MVC? architecture stands for Model View Controller which is the idea that the Model takes care of the data that the user sees through the View and then inputs through the Controller which gives the data back to the Model. An infinite loop.

10. What is the difference between a service and a factory? A service is by definition an object while a factory is a funciton.
